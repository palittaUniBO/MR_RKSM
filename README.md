This software package includes the minimal residual rational Krylov subspace method for sequences of shifted linear systems developed and analysed in the manuscript

Minimal Residual Rational Krylov Subspace Method for Sequences of Shifted Linear Systems. Hussam Al Daas, Davide Palitta. ArXiv: 2507.00267.

This software is intended to reproduce some the numerical experiments illustrated in the manuscript mentioned above. The software has been developed by using Matlab, version 2022b.
